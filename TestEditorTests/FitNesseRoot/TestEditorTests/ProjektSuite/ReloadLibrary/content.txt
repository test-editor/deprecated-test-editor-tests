This test cases checks, whether the reload library button in the project settings page (Right  Click on any project node in the main tree) works ok.
The tests makes a temporary change to the action group configuration, i.e. creates a new dummy actiongroup/mask ("NewActionGroup"). It than triggers the reload button and checks afterwards, if the new mask appears in the test step dialog view.
Finally the action group entry is removed from the configuration.
